El error que estás viendo es debido a un conflicto de dependencias entre sass-loader y webpack. sass-loader@7.3.1 requiere webpack@^3.0.0 || ^4.0.0, pero tienes webpack@5.95.0 instalado.

Para resolver este problema, puedes actualizar sass-loader a una versión compatible con webpack@5.x. Aquí tienes los pasos:

Actualiza sass-loader a una versión compatible con webpack@5.x. La versión más reciente de sass-loader debería ser compatible.
npm install sass-loader@latest --save-dev

Luego, intenta instalar las dependencias nuevamente.
npm install

En caso de tener errores
npm audit fix
npm fund
